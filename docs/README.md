### 文档贡献指南

#### 文档编写规范
1. **格式规范**：使用Markdown格式，遵循统一的标题层级
2. **内容规范**：确保内容准确、完整、易懂
3. **更新规范**：及时更新文档，保持与代码同步
4. **审核规范**：重要文档变更需要经过团队审核

#### 文档结构规范
```
docs/
├── README.md                 # 文档中心首页（本文件）
```
